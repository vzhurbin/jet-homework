## Установка

Склонируйте или загрузите репозиторий

```Shell
$ git clone https://github.com/vzhurbin/jet-homework
```

Установите зависимости командой

```shell
  yarn install
```

или

```shell
  npm install
```

Запустите `yarn` или `npm` с командой `start` для запуска dev-сервера

```Shell
yarn start
```

или

```Shell
npm run start
```
